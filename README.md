# Quversi (Quantum Reversi)

**量子リバーシ「Quversi」のソースコードリポジトリです。**

## 🎮 ゲームをプレイ
**[ここをクリックしてゲームをプレイ (GitHub Pages)](https://tomonta.github.io/quantum-reversi/)]**

## ✨ 特徴
- **量子重ね合わせ**: 70%や90%の確率で色が変わるコマが登場。
- **観測**: 「観測」ボタンで盤面が確定し、大逆転のチャンス。
- **50%コマ (ジョーカー)**: 4ターン目以降、相手の石を利用できる強力なジョーカーコマが登場。
- **オンライン対戦**: リアルタイムで対戦が可能。
- **CPU対戦**: 初級・中級・上級AIと対戦可能（上級はアルファベータ探索を実装）。

## 🛠️ 技術スタック
- HTML5 / CSS3 / Vanilla JavaScript
- Firebase (Realtime Database & Authentication) for matchmaking & sync
